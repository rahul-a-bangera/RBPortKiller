================================================================================
                   RBPortKiller - Port Management CLI Tool
================================================================================

WHAT IS IT?
-----------
View and manage network ports and their associated processes.
Identify which programs are using specific ports and terminate them.

INSTALLATION
------------
Automatic Setup (Recommended):
  1. Run: .\install.ps1 in PowerShell
     (Right-click PowerShell > "Run as Administrator" for system-wide install)
  2. Restart your command prompt/PowerShell
  3. Type 'rbportkiller' from any directory

Manual Setup:
  - Run directly: .\rbportkiller.exe
  - Or add this folder to your system PATH

USAGE
-----
Type: rbportkiller

Interactive menu options:
  1. View all active network connections
  2. Filter by port number
  3. Kill processes using specific ports

REQUIREMENTS
------------
Windows 10/11 (x64 or x86) - Self-contained, no dependencies
